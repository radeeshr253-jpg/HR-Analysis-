# HR Analytics Dashboard

This is my Power BI dashboard for HR Analytics.  
I made this project to understand employee attrition and basic HR insights.

The dashboard shows:
- Total employees
- Total attrition
- Attrition rate
- Age and salary details
- Department-wise attrition
- Education-wise attrition
- Job role information
